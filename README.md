# Studio Niza

A one-person Shopify growth studio helping new and growing Shopify owners turn visitors into customers.

🌐 **Live site:** [studioniza.com](https://studioniza.com)

## What Studio Niza Offers

- **Shopify SEO & Listing Optimization.** Product, collection, and page optimization with keyword research, schema, and indexing follow-up.
- **AI Chatbots for Shopify.** Custom-built bots that answer FAQs, recover abandoned carts, and capture leads.
- **SEO Blog Content.** Researched, written, and published blog posts that build organic traffic over time.
- **Reviews Management.** Automated review collection, response, and the migration of reviews from discontinued listings to active ones.

## Tech Stack

- **Hosting:** Cloudflare Workers (Static Assets)
- **DNS & Security:** Cloudflare
- **Email:** Zoho Mail
- **Forms:** Tally
- **Analytics:** Cloudflare Web Analytics

## Project Structure

This is a static HTML site, deployed automatically to Cloudflare on every push to `main`.
